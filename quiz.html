# ‡∏ï‡∏±‡∏ß‡∏≠‡∏¢‡πà‡∏≤‡∏á‡∏Å‡∏≤‡∏£‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏ä‡πà‡∏ß‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏ä‡∏∑‡πà‡∏≠‡∏°‡∏±‡πà‡∏ô
calculate_ci <- function(mean_sample, sd_sample, n, confidence_level) {
    # ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì Z-score
    alpha <- 1 - confidence_level
    z_score <- qnorm(1 - alpha/2)
    
    # ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì Standard Error
    SE <- sd_sample / sqrt(n)
    
    # ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏ä‡πà‡∏ß‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏ä‡∏∑‡πà‡∏≠‡∏°‡∏±‡πà‡∏ô
    lower_bound <- mean_sample - z_score * SE
    upper_bound <- mean_sample + z_score * SE
    
    # ‡πÅ‡∏™‡∏î‡∏á‡∏ú‡∏•‡∏•‡∏±‡∏û‡∏ò‡πå
    cat("‡∏Ñ‡πà‡∏≤‡πÄ‡∏â‡∏•‡∏µ‡πà‡∏¢:", mean_sample, "\n")
    cat("‡∏£‡∏∞‡∏î‡∏±‡∏ö‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏ä‡∏∑‡πà‡∏≠‡∏°‡∏±‡πà‡∏ô:", confidence_level * 100, "%\n")
    cat("Z-score:", round(z_score, 3), "\n")
    cat("Standard Error:", round(SE, 3), "\n")
    cat("‡∏ä‡πà‡∏ß‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏ä‡∏∑‡πà‡∏≠‡∏°‡∏±‡πà‡∏ô: (", round(lower_bound, 2), ", ", 
        round(upper_bound, 2), ")\n")
    
    return(c(lower_bound, upper_bound))
}

# ‡∏ï‡∏±‡∏ß‡∏≠‡∏¢‡πà‡∏≤‡∏á‡∏Å‡∏≤‡∏£‡πÉ‡∏ä‡πâ‡∏á‡∏≤‡∏ô - ‡πÇ‡∏à‡∏ó‡∏¢‡πå‡∏ó‡∏µ‡πà 1
result <- calculate_ci(12.4, 3.2, 60, 0.99)
